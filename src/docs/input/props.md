## Props

| props       | description          | type                | default    | required |
| --------    | -------------------- | ------------------- | ---------- | -------- |
| value       | 输入框内容            | string              | ''         | true     |
| onChange    | 输入框内容变化时的回调 | (e: string) => void | () => void | true     |
| label       | 标签文本              | string              | -          | false    |
| placeholder | 输入框是否不可用       | string              | -          | false    |
